# \# Local LLM Deployment on Edge Hardware

# 

# Deploy AI models locally — no cloud, no API key, no data egress.

# Built for healthcare, finance, and defense where data privacy is critical.

# 

# \## What This Project Does

# \- Runs a local AI model on your computer

# \- Summarises patient notes like a clinical assistant

# \- Zero data leaves your machine
